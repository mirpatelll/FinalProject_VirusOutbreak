📋 How to grade this app
  1. Sign in with any callsign (letters, numbers, underscores). New names auto-register.
  2. By default you're connected to our server. Click "Join different server" below to test cross-team interoperability — paste any team's API base URL and click Use.
  3. Create or join a game from the lobby. Grid size 5-15, max players 2-10.
  4. Place your ships (4 ships: submarine, destroyer, cruiser, battleship) by clicking cells.
  5. Fire by clicking your opponent's board on your turn. Hit/miss/sunk shown in real time.
  6. Note: our server is on Render free tier — first request after idle may take ~30s to wake up.
  7. Light/dark mode toggle is at the top right.
